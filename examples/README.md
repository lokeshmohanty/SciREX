# SciREX Examples

This directory contains various examples demonstrating the capabilities of the `SciREX` framework across different domains.

## Installation

To run these examples, you may need to install optional dependencies:

```bash
# For simple examples
pip install scirex[eg-simple]

# For experimental examples
pip install scirex[experimental]
```

## File Structure

```
.
├── simple/         # Simple, end-to-end examples demonstrating core SciREX modules.
└── experimental/   # Research-focused implementations and experimental features.
```

- **[Simple Examples](./simple/README.md)**: Beginner-friendly examples for core modules.
- **[Experimental Examples](./experimental/README.md)**: Advanced research and experimental workflows.
