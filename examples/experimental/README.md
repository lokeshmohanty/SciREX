# Examples: Experimental

This directory contains research-focused implementations and experimental features. These examples demonstrate advanced workflows and are subject to frequent changes.

## File Structure

```
.
├── dl/                          # Deep Learning experiments.
├── ml/                          # Machine Learning experiments.
├── model_compression/           # Pruning and quantization research.
├── model_reduction_pipeline/    # Scientific model reduction.
├── sciml/                       # Scientific ML (PINNs, etc.).
└── v2/                          # Experimental version 2 components.
```

## Detailed Overview

- **[Deep Learning (dl)](./dl/)**: Core research in neural network architectures and training methodologies.
- **[Machine Learning (ml)](./ml/)**: Classical machine learning experiments and utility implementations.
- **[Model Compression](./model_compression/)**: Research into pruning, quantization, and architectural efficiency.
- **[Model Reduction Pipeline](./model_reduction_pipeline/)**: Specialized workflows for scientific model reduction.
- **[Scientific ML (sciml)](./sciml/)**: Implementation of Physics-Informed Neural Networks (PINNs) and related methods.
- **[Experimental v2](./v2/)**: Pre-release version 2 components and architecture tests.

## Disclaimer

The code in this directory is experimental. API stability and performance guarantees may not apply.
